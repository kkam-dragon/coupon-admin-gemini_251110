## 1️⃣ 개요

본 문서는 **㈜티사이언티픽 모바일커머스 시스템 연동 시 리턴되는 Result Code 명세서**이다.

### ⚙️ 시스템 정보

| 항목 | 내용 |
|:----|:----|
| **상용 URL** | `https://corp.coufun.kr:446/b2c_api/` |
| **테스트 URL** | `http://tcorp.coufun.kr/b2c_api/` |
| **통신방식** | HTTP `GET` 또는 `POST` |
| **결과형식** | XML |

### 📡 주요 연동 API 목록

- 상품목록, 상품정보 조회  
- 쿠폰발송  
- 쿠폰생성  
- 쿠폰취소  
- 쿠폰재발송  
- 쿠폰수신자번호변경  
- 쿠폰상태조회  
- 쿠폰이미지전송  

### ⚠️ 유의사항

1. 반드시 **테스트 서버에서 연동 구현 후 상용 서버 적용**  
2. **IP 등록 필수**, 미등록 시 사전 요청 필요

## 2️⃣ 결과 코드 명세

| 코드 | 메시지 | 설명 |
|:----:|:--------|:------|
| 00 | 처리 성공 | 정상적으로 실행 완료 |
| 01 | IP 오류 | 등록되지 않은 IP로 요청한 경우 |
| 02 | POC_ID 오류 | 등록되지 않은 또는 사용중단된 클라이언트 ID로 요청 (관리자 문의) |
| 03 | ORDER_ID 오류 | 잘못된 주문 ID로 요청 |
| 04 | 유효기간 오류 | 유효기간이 지난 쿠폰 요청 |
| 05 | 연동기간 오류 | 연동기간이 지난 경우 |
| 08 | 상품수량 오류 | 상품 수량이 잘못된 경우 |
| 09 | 상품목록 조회 오류 | 일일 조회 횟수 초과 |
| 10 | 상품정보 오류 | CAT_ID, GOODS_ID 정보가 잘못된 경우 |
| 11 | TR_ID 사이즈 오류 | 입력 가능한 길이(50byte) 초과 |
| 12 | TR_ID 중복 오류 | 기존 요청한 TR_ID로 다시 요청 |
| 13 | 발신번호 오류 | 발신번호가 숫자가 아니거나 12byte 초과 |
| 14 | 수신번호 오류 | 수신번호가 숫자가 아니거나 휴대폰번호 형식이 아님 |
| 15 | 수신번호 개수 초과 | 10개 이상의 수신번호 요청 |
| 16 | 쿠폰번호 오류 | 쿠폰번호가 잘못된 경우 |
| 17 | 사용된 쿠폰 | 이미 사용된 쿠폰 요청 |
| 18 | 취소된 쿠폰 | 이미 취소된 쿠폰 요청 |
| 19 | 발송타입 오류 | 잘못된 발송 요청 |
| 20 | 발송횟수 초과 | 발송 횟수 제한 초과 (3회 제한) |
| 21 | 제목 오류 | 제목 미입력 또는 40byte 초과 |
| 22 | 내용 오류 | 내용 300byte 초과 |
| 23 | 중복 요청 오류 | 동일 요청 반복 수행 |
| 24 | 추가입력1 오류 | 입력 길이 초과 (200byte) |
| 25 | 추가입력2 오류 | 입력 길이 초과 (200byte) |
| 26 | 유효기간 설정 오류 | 유효기간 입력 오류 |
| 27 | 유효기간 종료일 오류 | 종료일 입력 오류 |
| 28 | 유효기간 오류 | 유효기간 변경 불가 상품 |
| 29 | 유효기간 시작일 오류 | 시작일 입력 오류 |
| 30 | 판매상품 사용기간 오류 | 상품 사용 가능 기한 아님 |
| 31 | 판매상품 ID 오류 | 유효하지 않은 판매상품 ID |
| 50 | 쿠폰생성 오류 | 쿠폰 생성 실패 (관리자 문의) |
| 95 | 발행 한도 오류 | 발행 한도 초과 (관리자 문의) |
| 98 | API 권한 오류 | 사용 불가 기능 호출 |
| 99 | 기타 오류 | 시스템 오류 (관리자 문의) |
